## 📊 考古題分析（五卷）

- 新舊考卷考點**重複率約 82%**（題型相同、多半換數字）。
- **最穩定必考**（五卷幾乎都出）：bitwise-AND 子網、反平方律、子網位元數計算（2ᴺ−2）、路由表最長前綴、TTL/Hop Limit、PSK、MPLS、IPsec 模式、五大傳播問題、CSMA。
- **新卷新增考點**：IPv4 header length=5、TCP Source/Dest Port、Identification/Flags/Fragment Offset、T1 bonding 速度、Service Band vs Channels、Channel Reuse、SMTP/POP/IMAP/RFC822 解釋、Flow Label→IPv6 / Protocol→IPv4。

## ✅ 使用建議

1. 從 `index.html` 進入，先掃各章頂部的 **🔥 出題頻率表** 鎖定必考。
2. 看 **🆕 新卷考點專區** 補齊 2023/2025 新增題型。
3. 用各章「**考前速記表**」＋ `formula.docx` 記住計算步驟。
4. 拿兩份「題目與解答」實作，再用每章 **10 題測驗** 自測。
