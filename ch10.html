2023 / 2025 新卷考點重點New Exams — 重複考點＋新增考點講解

🔁 Repeated　舊卷考過、新卷再考

  • ADSL：2025 A7、2023 A13 都考「1-pair VG UTP、下行高上行低」＝ADSL。
  • MPLS：2025 A11、2023 A12 都考「在 IP header 前插 label、依 label 轉送」＝MPLS。
  • Leased line 例子：2023 A14 → T1（1.544 Mbps）。

🆕 T1 / bonded T1 / T3 速度比較　bonding（2025 A12）🆕 新卷

  • Bonding（綁定）：T1/E1 線可綁在一起提供 2、3、4 倍容量。
  • 速度：T1 = 1.544 Mbps；五條 bonded T1 = 5 × 1.544 = 7.72 Mbps；T3 = 44.736 Mbps。
  • ∴ 最快是 T3（44.736 > 7.72 > 1.544）。
📄 【2025 A12】T1、five bonded T1、T3 三者誰最快？2025
▶ 看答案
T3（44.736 Mbps）。五條 bonded T1 只有 7.72 Mbps，仍遠小於 T3。

🆕 Channel Reuse　頻道重用名詞解釋（2023 B1）🆕 新卷

  • 在蜂窩網路中，於不同 cell 重複使用相同頻道，使營運商能服務更多客戶。
  • 例：500 頻道 × 100 個 cell = 50,000 個同時用戶（沒重用只能 500）。
📄 【2023 B1(3)】Explain Channel Reuse (for Cellular Telephone Service).2023
▶ 看答案
在不同 cell 重複使用相同頻道，使營運商能以同一頻道服務多個客戶（500 頻道 × 100 cell = 50,000 同時用戶）。

🆚 Internet vs 企業 WAN
Internet：無所不在又便宜，但缺乏組織所需的 QoS；企業 WAN：提供所需 QoS，但比 Internet 貴很多，主要用於公司各站點之間的傳輸。

🔌 Network Access vs Network Core　接取 vs 核心

  • Network Access（接取）：用戶端設備（CPE）經 access link（接取線路） 連到電信業者網路。
  • Network Core（核心）：電信業者內部用來互連各站點的高品質傳輸網路。

ADSL 與纜線數據機Residential Access — ADSL & Cable Modem

📞 ADSL　非對稱數位用戶線路🔥 考過2次

  • 本地迴路（local loop）原本就把 一對（1-pair）語音級（voice grade）UTP 拉到每一戶；原本不是設計來傳資料。
  • ADSL 用進階技術，讓資料能跑在這條既有的 1-pair VG 線上。
  • 💰 因為線路早就裝好了、不必再拉新線，所以 ADSL 便宜。
  • Asymmetric（非對稱）：下行（downstream）高、上行（upstream）受限。
  • 機房端設備：DSLAM（DSL Access Multiplexer）。
📄 【109-1 A10】____ provides residential Internet access over one-pair voice-grade UTPs, high downstream, limited upstream.109-1
▶ 看答案
ADSL（Asymmetric DSL；利用既有 1-pair VG UTP，下行高上行低）。

📺 Cable Modem Service　纜線數據機

  • 原本是為了把電視訊號送到收訊不良的家庭而生（同軸電纜 coaxial cable），現在也提供雙向資料服務。
  • 在美國很普及，但多數國家不普及。
🧩 ADSL vs Cable
Cable：個人吞吐量「稍快」、成本「稍高」；ADSL：吞吐量「稍慢」、成本「稍低」。
3

行動電話系統Cellular Telephone Service

📱 Cell, Cellsite, Channel Reuse, Handoff　五個關鍵詞考過 (handoff)

  • Cell（細胞）＝一塊地理區域；Cellsite（基地台）＝一棟放行動設備的建築。兩者別搞混。
  • Channel Reuse（頻道重用）：同一頻道可在不同 cell 重複使用 → 一個頻道能服務多個客戶。
    例：500 頻道 × 100 cells → 可服務 50,000 個同時用戶（沒重用只能 500）。
  • Handoff（換手 / Roaming）：用戶在 cell 之間移動時，被「交接」到新 cell 的 cellsite。
📄 【卷三 A6】In cellular telephony, ____ occur when a subscriber moves between cells.考古
▶ 看答案
handoffs（換手 / roaming）。

🏢 MTSO & LTE　行動交換局與速度

  • MTSO（Mobile Telephone Switching Office，行動電話交換局）：連接各 cellsite，並接到 PSTN（公眾電話網），負責換手與通道分配。
  • cellsite 用點對點微波天線連到 MTSO。
  • 現今多用 LTE（Long-Term Evolution）；尖峰下載多為 5–20 Mbps；LTE Advanced 要求 3 Gbps 下行。
  • 3G/4G/5G 世代由 ITU 定義，但行銷常浮濫使用。
📄 【卷三 B13(a)】Briefly explain MTSO.考古
▶ 看答案
MTSO：行動電話交換局，連接各 cellsite 並接到 PSTN，負責換手與通道分配。

專線 Leased LinesNetwork Core — Leased Lines

🔗 Leased Line & POP　專線與接取點考過

  • Leased line（專線）屬 Layer 1，可當 access line。
  • 連到 ISP 最近的 POP（Point of Presence，接取點）。每個站點用一條專線當 access line 連到當地 POP。
  • 每個客戶站點需 CSU/DSU（Channel/Data Service Unit） 介面連到電話網路。

三層核心技術

技術層特性
Leased LinesLayer 1專線、access line
Carrier EthernetLayer 2把站點乙太網路延伸到整個公司
MPLSLayer 3強流量工程、高 QoS
📄 【109-1 A8】To connect all sites via PSDN, each site uses a ____ line as access line to ISP's nearest POP.109-1
▶ 看答案
Leased line（專線；當作 access line 連到最近的 POP）。

Carrier EthernetSwitched (Layer 2) Technology

🔀 E-Line & E-LAN　兩種服務

  • E-Line service：像擁有一條專線（點對點）。
  • E-LAN service：把站點的乙太網路 LAN 延伸到多個站點。
  • 歷史 PSDN 技術（Frame Relay、ATM）已凋零；現在主流是 Carrier Ethernet。

Carrier Ethernet 好處

  • 低成本、員工熟悉的技術、可用高速度、速度敏捷（speed agility，可快速升速）、QoS、安全。

MPLS 標籤交換Multiprotocol Label Switching

🏷️ Why MPLS?　為何要 MPLS🔥 考過2次

  • 傳統路由：每個封包都要走「找符合列→挑最佳→送出」三步驟，很耗處理能力 → 貴。
  • MPLS：先為一段 IP 位址範圍找好最佳路徑，給它一個 label number（標籤號碼）；之後每個封包帶這個 label。
  • 路由器只做快速查表（用 label 查），不必重跑三步驟 → 處理量小、比傳統路由便宜。
  • label 位置：夾在 frame header 與 IP packet header 之間。
MPLS Label 夾在 Frame Header 與 IP Packet Header 之間
📄 【109-1 A9】____ is a routing mechanism. It inserts a label in front of the IP header.109-1
▶ 看答案
MPLS（Multiprotocol Label Switching；以 label 查表轉送，較傳統路由便宜）。
📄 【2020 A7】____, making routing more efficient. A label header is inserted in front of the IP header.2020
▶ 看答案
MPLS（Multiprotocol Label Switching）。

PSDN 廣域網方案Recommending a WAN Solution

🗺️ Three-Site WAN via PSDN　三站互連方案考過 (簡答)

【2020 B3】NCNU 有三站（台中、台南、台北）要用 WAN 互連，中華電信提供專線與三市都有 POP 的 PSDN。

  • 建議用 PSDN（如 MPLS / Carrier Ethernet）：每個站點只需一條專線當 access line 連到當地最近的 POP，再由電信業者的 PSDN 雲把三站互連。
  • 💰 比起三站兩兩拉專線（全網狀 N 站需 N(N−1)/2 條，3 站需 3 條）成本低很多，且具 QoS。
三站各拉一條專線到當地 POP，由 PSDN 雲互連（比全網狀省線又有 QoS）
📄 【2020 B3】NCNU 三站經 WAN 互連，推薦方案並畫圖。2020
▶ 看完整解答
建議用 PSDN（MPLS/Carrier Ethernet）：每站只需一條專線當 access line 連到當地最近 POP，再由電信業者 PSDN 雲互連三站。比三站兩兩拉專線（全網狀 N(N−1)/2 條）成本低很多，且具 QoS。圖：台中／台南／台北三站各→專線→各自 POP→中華電信 PSDN 雲互連。

互動測驗Self-Check Quiz — 點選作答，立即看詳解

🧪 隨堂測驗（10 題）

1. 利用既有「一對語音級 UTP」提供家用上網、下行高上行低的技術是？
✅ ADSL（Asymmetric DSL）：用既有 1-pair VG UTP，下行高、上行受限，因不必拉新線而便宜。
2. 「在 IP 標頭前面插入一個 label、以查表方式轉送、比傳統路由便宜」的是？
✅ MPLS：先為一段位址找好路徑給 label，之後只快速查表，省處理量。
3. 把 T1、ADSL(10Mbps下行)、T3、802.11g 由慢到快排序，正確的是？
✅ T1=1.544 < ADSL=10 < T3=44.736 < 802.11g=54 Mbps。
4. MPLS 的 label 位於哪個位置？
✅ MPLS Label 夾在 frame header 與 IP packet header 之間。
5. 用戶在 cell 之間移動時發生的是？
✅ handoff（換手/roaming）：被交接到新 cell 的 cellsite。
6. 連接各 cellsite、並接到 PSTN、負責換手與通道分配的是？
✅ MTSO（Mobile Telephone Switching Office，行動電話交換局）。
7. 每個站點用一條 ____ 當 access line 連到 ISP 最近的 POP。
✅ Leased line（專線） 當 access line 連到最近 POP；站點需 CSU/DSU 介面。
8. 三站要互連，用 PSDN 比「兩兩拉專線」好的主因是？
✅ PSDN：每站只需一條專線到當地 POP，由雲互連；全網狀需 N(N−1)/2 條，成本高。PSDN 省線又有 QoS。
9. 關於 cell 與 cellsite，何者正確？
✅ cell＝地理區域；cellsite＝放行動設備的建築。
10. 500 個頻道、100 個 cell，搭配頻道重用最多可服務幾個同時用戶？
✅ 頻道重用：500 × 100 = 50,000 個同時用戶（沒重用只能 500）。
